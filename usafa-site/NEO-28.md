# Add "Review USAFA Webpage with Dad" Task to Neo Dashboard

---

## Summary

Add a personal review task to the Neo dashboard that prompts a joint session with Dad to evaluate the USAFA webpage.

---

## Change

### Neo Dashboard — New Task Card

Add the following task entry to your Neo dashboard task list:

```markdown
## Task: Review USAFA Webpage with Dad

- **Title:** Review USAFA Webpage with Dad
- **Priority:** Medium
- **Due Date:** [Set your target date]
- **Tags:** `usafa` `review` `family` `web`
- **Status:** To Do

### Checklist
- [ ] Schedule a time with Dad to sit down together
- [ ] Open the USAFA webpage side by side on screen
- [ ] Walk through each section and note anything that looks off
- [ ] Check that links and buttons work correctly
- [ ] Note any content, photos, or wording Dad thinks should change
- [ ] Capture all feedback in writing before the session ends
- [ ] Bring notes back here for next steps
```

---

## Notes

- **No code ships** from this task — it is purely a review and feedback-gathering session.
- Keep a notes doc open during the review so nothing Dad flags gets forgotten.
- If Dad spots something he wants changed, that becomes its own follow-up task with specific details filled in.
- Once feedback is collected, return here and a proper change draft can be written up for approval before anything touches the live site.
